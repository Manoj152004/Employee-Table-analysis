## Employee Data Analysis

Analyzed an employee dataset using SQL to extract HR and compensation insights.

**Tools:** MySQL  
**Concepts used:** Aggregate functions, Subqueries, CTEs, JOINs, Window functions (ROW_NUMBER, DENSE_RANK), CASE statements, GROUP BY

**Key findings:**
- Identified employees earning above average salary by department
- Ranked employees using ROW_NUMBER and DENSE_RANK across departments
- Classified workforce into salary brackets (Above Average / Average / Adjustable / Low)
- Counted employee distribution across departments

**File:** `employee_analysis.sql`
